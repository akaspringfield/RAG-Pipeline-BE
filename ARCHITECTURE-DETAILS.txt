Tech Responsibilities Map
Layer	Tech
API Layer	Flask + Gunicorn
Auth	JWT
RBAC	DB + Middleware
Cache	Redis
DB	PostgreSQL
AI	External LLM / RAG engine
Reverse Proxy	Nginx
Deployment	Docker

🚀 Key Design Highlights
✔ RBAC is DB-driven

No hardcoded permissions

✔ AI layer is isolated

Chat service never talks directly to DB logic

✔ Stateless backend

JWT + Redis optional session caching

✔ Scalable

Can move to Kubernetes easily
